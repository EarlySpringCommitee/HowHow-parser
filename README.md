# HowHow-parser
HowHow 影片處理器，協助分割每個發音
## 使用方法
請先安裝 ffmpeg 與 node.js！
```bash
npm i # 第一次使用
node index.js # 可能會跑很久很久
```
## 發音修正
請修改 Repo 裡的 `original/HowHow 發音標註眾包 - 發音表.csv`

## 下載發音集
